# Normative Test Vectors

These 10 JSON files contain the exact, normative test vectors for CSP-Core
v2.1.2. Each file includes:

- `artifact_hex`: the artifact bytes as hex
- `t_in`, `t_out`: UTC timestamps
- `label`: logical location label
- `csl_canonical`: the canonical JSON string of the CSL for hashing into S2
- `s1`: SHA-384 of the artifact as lower-case hex
- `s2`: SHA-384 of `canonical_CSL || 0x1f || s1` as lower-case hex
- `bangcheck`: a BangCheck metadata block for the vector

All hashes were generated by the normative Python reference implementation
`impl/python/csp_core_v2_1_2.py`.

## Conformance

Any implementation that reproduces all 10 `s2` values byte-for-byte is
considered conformant. See `/CONFORMANCE.md`.
